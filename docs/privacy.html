Privacy Policy

Datasurf Chrome Extension — published by Io Recupero SRL

Data collection

This extension does not collect, transmit, store, or share any personal data or usage information of any kind.

How it works

The extension operates entirely within the user's browser. It reads DOM content and intercepts XHR responses on https://app.datasurf.it solely to inject additional UI controls (filters and lookup cards) into the existing interface. No data leaves the user's device as a result of this extension.

Third-party services

The extension does not communicate with any server or third-party service. It contains only static lookup tables (European Waste Catalogue codes) bundled in the extension package itself.
